# DeepLink

Инструкция по использованию сайта:

Основные функции:

1. Сокращение ссылок: Вставьте длинный URL в поле ввода и нажмите "Сократить ссылку"
2. Пользовательский псевдоним: При желании укажите свой вариант короткой ссылки
3. Копирование результата: Скопируйте полученную короткую ссылку с помощью кнопки "Копировать"
4. Создание новой ссылки: После создания ссылки можно создать новую с помощью кнопки "Новая ссылка"

Технические особенности:

· Полностью адаптивный зеленый дизайн
· Подготовленная структура для БД
· Модульная архитектура для легкой кастомизации
· Чистый и комментированный код

Сайт готов к использованию. Для полноценной работы необходимо реализовать backend-часть с подключением к вашей базе данных.
